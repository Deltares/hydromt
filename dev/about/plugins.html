Plugins#

HydroMT model plugins#

HydroMT is commonly used in combination with a model plugin which provides a HydroMT implementation for specific model software. Using the plugins allows to prepare a ready-to-run set of input files from raw geoscientific datasets and analyse model results in a fast and reproducible way.

Known model plugins include:

Wflow plugin

https://deltares.github.io/hydromt_wflow/

SFINCS plugin

https://deltares.github.io/hydromt_sfincs/

Delwaq plugin

https://deltares.github.io/hydromt_delwaq/

Delft-FIAT plugin

https://deltares.github.io/hydromt_fiat/

Delft3D FM plugin

https://deltares.github.io/hydromt_delft3dfm/

About the model software#

  • Wflow is Deltares’ solution for modelling hydrological processes, allowing users to account for precipitation, interception, snow accumulation and melt, evapotranspiration, soil water, surface water and groundwater recharge in a fully distributed environment.

  • SFINCS is Deltares’ reduced-complexity model, designed for super-fast modelling of compound flooding events in a dynamic way.

  • Delwaq is Deltares’ water quality process library used to quantify point source and diffuse emissions (D-Emissions) and model the fate and transport (D-Water Quality) of various substances/pollutants in surface water systems.

  • Delft-FIAT is Deltares’ Flood Impact Assessment Toolbox and used to quantify the impact and risk of flooding and other perils.

  • Delft3D-FM is Deltares’ Delft3D Flexible Mesh Suite (Delft3D FM) for hydrodynamic modelling

Starting your own plugin#

If you would like to start your own HydroMT plugin for another model or software, we have some tips and a quick start page for you: Starting Your Own HydroMT Plugin.

If you would like your plugin to appear on this page, please contact us via the issue board of HydroMT!
