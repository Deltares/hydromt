Examples: Data Catalog
======================

HydroMT provides a data catalog system to easily access and manage various types of
hydrological and geospatial data. The following examples demonstrate how to prepare a
data catalog, export data for a certain region, and read different types of data
(raster, vector, point, tabular) using HydroMT.

Please see the :ref:`installation guide <installation_guide>` to see how to install HydroMT
and to :ref:`download the examples <installation_examples>`.
