Data conventions#

Names and units mentioned here are mandatory in order for the input data to be processed correctly and produced the right derived data. It is possible to use the rename option in the data catalog yaml file so that data variables have hydroMT-compatible names. This section lists the different variable naming and unit conventions of HydroMT by types. A list of recognized dimension names is found here.
